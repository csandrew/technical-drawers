# Technical Drawers

Engineering & Architecture Supplies Website

## About
Technical Drawers supplies quality equipment to engineering and architecture students in Kenya. This website showcases products and directs customers to WhatsApp for inquiries.

## Features
- Year-based navigation for students
- Product showcase with WhatsApp integration
- Resource guides for students
- Mobile-first responsive design

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome (icons)

## Setup
1. Clone the repository
2. Open `index.html` in your browser
3. Update WhatsApp number in all files
